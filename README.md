# vibeverse-cli

Official command-line interface for the Vibeverse ecosystem.

## Scope

- Wrap `vibeverse` engine workflows in a friendly CLI.
- Support composition, catalog tooling, and pack management.
- Provide deterministic generation and offline-first workflows for game dev use.

## Planned Package

- npm: `vibeverse-cli`
- binary: `vibeverse`
